### Deliverables

- URL for the final code of the front-end app in a public Github repository
- The URL of a deployed version of the app in Netlify or a similar free hosting service mentioned in readme.md

Use the following REST APIs to fetch the data (as you are not expected to build a server in the frontend assessment)

- Albums API:  [https://jsonplaceholder.typicode.com/albums](https://jsonplaceholder.typicode.com/albums)
- Songs API: [https://jsonplaceholder.typicode.com/photos](https://jsonplaceholder.typicode.com/photos) (The photos API contains the list of songs in each album along with its thumbnail image)
- The URL Params and queries information can be found in [https://jsonplaceholder.typicode.com/](https://jsonplaceholder.typicode.com/)
